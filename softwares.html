Softwares

Lista de softwares

R é um ambiente de software livre para computação e gráficos estatísticos. Compila e roda em uma ampla variedade de plataformas UNIX, Windows e MacOS.

  • Software R

O RStudio é um conjunto de ferramentas integradas projetadas para ajudá-lo a ser mais produtivo com o R. Inclui um console, editor de realce de sintaxe que suporta execução direta de código e uma variedade de ferramentas robustas para plotagem, exibição de histórico, depuração e gerenciamento de seu espaço de trabalho.

  • RStudio

O Anaconda Individual Edition de código aberto (formalmente Anaconda Distribution) é a maneira mais fácil de realizar ciência de dados Python / R e aprendizado de máquina no Linux, Windows e Mac OS X.

  • Anaconda3 v5.2

O Sublime Text é um software multiplataforma de edição de texto, no entanto utilizado por muitos desenvolvedores para editar código-fonte, escrito em linguagem Python.

  • Sublime Text

PyCharm é um ambiente de desenvolvimento integrado usado em programação de computadores, especificamente para a linguagem Python.

  • PyCharm

O Project Jupyter é um projeto de código aberto sem fins lucrativos, nascido do Projeto IPython em 2014, à medida que evoluiu para oferecer suporte à ciência de dados interativa e à computação científica em todas as linguagens de programação. O Jupyter sempre será um software 100% de código aberto, gratuito para uso de todos e lançado sob os termos liberais da licença BSD modificada .

  • jupyter

O JupyterLab é a interface de usuário baseada na Web de última geração do Project Jupyter.

  • jupyter Lab

O MiKTeX (pronuncia -se mick-tech ) é uma implementação atualizada do TeX / LaTeX e de programas relacionados. O TeX é um sistema de composição escrito por Donald Ervin Knuth, que diz que se destina à criação de belos livros - e especialmente a livros que contêm muita matemática.

  • MiKTeX

O TeXstudio é um ambiente de escrita integrado para a criação de documentos LaTeX. Nosso objetivo é tornar a escrita LaTeX o mais fácil e confortável possível. Portanto, o TeXstudio possui vários recursos, como realce de sintaxe, visualizador integrado, verificação de referência e vários assistentes.

  • TeXstudio
